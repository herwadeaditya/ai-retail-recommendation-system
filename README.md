# 🛍️ AI Retail Recommendation System

## 🚀 Live Demo
👉 https://ai-retail-recommendation-system.streamlit.app/

---

## 📌 Overview

The **AI Retail Recommendation System** is a smart data-driven application that helps retail shop owners increase sales using intelligent product recommendations and analytics.

It uses **Market Basket Analysis (Apriori Algorithm)** to identify frequently bought product combinations and provides actionable insights like demand trends, pricing strategies, and business decisions.

---

## 🎯 Features

### 🔥 Smart Product Recommendations
- Suggests frequently bought-together products
- Uses Apriori Algorithm (association rule mining)
- Displays **Lift & Confidence scores**

---

### 📊 Demand Analysis
- Categorizes products into:
  - 🔥 High Demand
  - ⚡ Medium Demand
  - 💡 Low Demand

---

### 💰 Revenue Insights
- Identifies top revenue-generating products
- Helps optimize pricing strategies

---

### 📦 Product Performance
- Highlights top-performing and low-performing products

---

### 🎯 Occasion-Based Intelligence
- Unified filtering for:
  - Seasons
  - Festivals
- Dynamic recommendations based on selected occasion

---

### 🎛 Interactive Dashboard
- Built using Streamlit
- Real-time charts and insights

---

## 🧠 Tech Stack

- Python 🐍
